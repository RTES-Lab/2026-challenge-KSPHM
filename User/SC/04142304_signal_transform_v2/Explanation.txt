== Signal Transformation v2 (2구간 이산화) ==
  RPM 분포: 저속(<850)=240회, 고속(≥850)=226회
  평균 Quality: 0.2446 → 0.2474 (Δ=+0.0028)
  개선: 45/76, 악화: 31/76

== HI 품질 (변환 후 상위 20개 → SPE) — 극적 개선! ==
  Bearing 1: Mon=0.080, Tre=0.935  ← Trendability 매우 높음
  Bearing 2: Mon=0.150, Tre=0.865
  Bearing 3: Mon=0.148, Tre=0.831
  Bearing 4: Mon=0.125, Tre=0.898
  ※ 이전 PCA SPE의 Trendability는 0.3~0.5 수준이었음 → > 0.83로 대폭 개선

== SSM (Kalman Filter + MC RUL) ==
  Bearing 1: FPT=20.83hr (마지막 직전), RUL=0.167hr
  Bearing 2: FPT=18.50hr (말기 구간),   RUL=0.167hr
  Bearing 3: FPT=13.67hr (전체의 93%),   RUL=0.378hr
  Bearing 4: FPT=16.67hr (전체의 74%),   RUL=5.872hr

  ※ sigma_w/sigma_v 비율이 크게 개선됨 (v1: sigma_w≈0, v2: 0.04~0.08)
    → Kalman filter가 관측값을 신뢰하기 시작 → 상태 추적 가능

[확인된 문제점]
  1. FPT가 여전히 말기에 집중
     - HI의 단조성(Mon)이 0.08~0.15로 아직 낮음
     - HI가 대부분 구간에서 0 근처 → 말기에 급상승하는 형태 지속
     - Trendability는 높지만 (Spearman 상관은 높음) 증가분이 편향되어 있음

  2. 특징(Feature) 수준에서 개선이 미미 (+0.3%)
     - 2구간 이산화에도 특징 자체의 품질 지표 개선이 적음
     - 원인: 개별 특징의 Signal Jump가 단순 선형(a'y+b')으로 설명 안됨
     - 특히 에너지 계열 특징은 RPM의 제곱에 의존 → 비선형 변환 필요 가능성
     - 그러나 HI(SPE) 레벨로 가면 Trendability가 0.83~0.94로 대폭 향상
       → PCA+SPE가 잔여 점프를 추가 흡수하는 효과

  3. Bearing 4의 불안정한 RUL
     - sigma_v가 매우 작고(0.013) sigma_w가 큼(0.082) → 과추정
     - 다른 3개 베어링과 열화 패턴이 상이 (고장 유형 차이 때문 추정)
