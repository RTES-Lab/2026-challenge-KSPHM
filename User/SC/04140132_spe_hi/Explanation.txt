[이전 실험]
04140103_initial_pca_result (PC1 Score 기반 HI)

[실험 목적]
PC1 Score 기반 HI가 말기에 감소하는 비단조(non-monotonic) 문제를 해결하기 위해,
같은 PCA 구조에서 "PC1 투영값" 대신 "SPE(재구성 오차)"를 HI로 사용하는 방식을 실험.

SPE(Squared Prediction Error, Q-statistic):
  - 정상 데이터로 학습된 PCA 모델이 해당 샘플을 얼마나 잘 설명하지 못하는지 측정
  - SPE = mean((x - x_hat)^2), x_hat: PCA 재구성값
  - 정상 구간: SPE 낮음 (PCA 모델이 잘 설명함)
  - 열화 구간: SPE 높음 (정상 공간에서 이탈)
  - PC1 Score와 달리 방향 의존성 없음 → 단조성 개선 기대

[이전 실험과의 차이점]
  PC1 Score: 특정 방향(PC1)으로의 투영값 → 열화 후반부 방향 이탈 시 감소 가능
  SPE:       투영 "후" 잔차의 크기 → 열화가 깊을수록 잔차 증가 → 단조성 우수

[데이터 특이사항 (기억 필요)]
  - Train1~4 베어링은 각각 다른 종류의 미세 고장이 실험 전 인가됨
  - 고장 유형(내륜/외륜/전동체 등) 미공개, 베어링마다 다를 수 있음
  - 가변속 조건(700~950 RPM, 1시간 단위 변동)
  - 다량의 환경 노이즈 포함

[실험 설정]
  - 특징 캐시: 04140103_initial_pca_result/output/ 재사용 (동일 76개 특징)
  - PCA 성분 수: 10 (이전 3에서 증가 → 더 많은 정상 패턴 포착, 잔차 공간 확보)
  - 정상 구간: 초기 15%
  - FPT 임계값: 정상 구간 μ + 3σ (SPE 기준)
  - HI 정규화: (SPE - normal_mean) / (max - normal_mean), clip [0,1]
  - 추가: HI_SPE_cm (cumulative max) → 단조 강제 버전도 함께 저장

[생성 파일]
  output/Bearing{1~4}_HI_SPE.csv     : SPE_raw, HI_SPE, HI_SPE_cm, FPT_sec
  output/Bearing{1~4}_HI_SPE.png     : SPE raw(log) + HI_SPE + HI_SPE_cm 3단 플롯
  output/All_HI_SPE_comparison.png   : 4베어링 비교 플롯
  output/HI_SPE_summary.csv          : 요약 통계

[확인된 문제점]
  1. FPT 임계값 붕괴 (μ+3σ ≈ 0)
     - 선형 PCA 10성분이 정상 분산의 약 99%를 설명
     - 잔차 공간(SPE가 측정하는 영역)이 0.2~1%에 불과
     - SPE 정상 구간 내 σ가 매우 작아 μ+3σ ≈ μ ≈ 0에 가까워짐
     - 결과: FPT 임계값이 사실상 0 → 정상 구간 초반(~3hr)에 바로 FPT 탐지
     - 실제 의미있는 열화 시작 시점과 무관한 조기 탐지 문제

  2. 특징 자체의 정상 구간 내 과도한 변동
     - 캐시된 features.csv는 raw(비정규화) 특징값
     - energy/kurt_rms 외 나머지 특징(64개)은 log 변환 없이 사용
     - 정상 구간에서도 가변속(700~950 RPM) 조건으로 인한 큰 변동 존재
     - → 76개 특징 중 노이즈성/중복 특징 제거 선행 필요

  3. 선형 PCA로는 비선형 열화 패턴 포착 한계 (KPCA로 해결 시도 예정)
