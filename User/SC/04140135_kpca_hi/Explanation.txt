[이전 실험]
04140132_spe_hi (선형 PCA + SPE 기반 HI)

[실험 목적]
선형 PCA의 한계를 극복하기 위해 KPCA(Kernel PCA, RBF 커널)를 적용.
커널 공간에서 3가지 HI를 동시에 비교:
  (A) KPCA PC1 Score
  (B) KPCA SPE (inverse_transform 기반 재구성 오차)
  (C) Mahalanobis Distance (커널 공간 내 정상 군집 중심으로부터의 거리)

[이전 실험에서 확인된 문제]
  선형 PCA 10성분이 정상 분산의 99%를 설명
  → 잔차 공간(SPE가 측정하는 영역)이 너무 작아짐(1% 미만)
  → FPT 임계값이 사실상 0 → 정상 구간 초반에도 FPT 탐지되는 과민 반응
  → 선형 모델로는 가변속/노이즈 환경의 비선형 열화 패턴 포착 불충분

[KPCA의 기대 효과]
  - RBF 커널이 특징 간 비선형 관계를 내재적으로 포착
  - 커널 공간에서 정상 데이터가 더 뭉치고, 이상 데이터가 더 분리됨
  - SPE와 MD 모두 더 명확한 정상/이상 구분 기대
  - 논문 Table 7 기준: KPCA Prognosability = 0.781 (최고값)

[데이터 특이사항 (기억 필요)]
  - Train1~4 베어링은 각각 다른 종류의 미세 고장이 실험 전 인가됨
  - 고장 유형(내륜/외륜/전동체 등) 미공개, 베어링마다 다를 수 있음
  - 가변속 조건(700~950 RPM, 1시간 단위 변동)
  - 다량의 환경 노이즈 포함

[실험 설정]
  - 특징 캐시: 04140103_initial_pca_result/output/ 재사용 (76개 특징)
  - 커널: RBF (gamma = 1/n_features = 1/76 ≈ 0.01316)
  - 성분 수: 10 (커널 주성분)
  - 정상 구간: 초기 15%
  - FPT 임계값: SPE기준 μ + 3σ (정규화된 HI 값)
  - cumulative max 버전도 함께 저장 (단조 강제 비교용)

[확인된 문제점]
  1. SPE FPT 임계값 여전히 붕괴 (선형 PCA와 동일 증상)
     - KPCA를 적용해도 SPE 정상 구간 mean이 이미 높음 (0.85~0.91)
     - std도 커서 μ+3σ가 max에 가까워짐 → 임계값 사실상 0
     - 원인: 76개 특징 중 정상 구간 자체에서 큰 변동을 가진 특징들 존재
     - 선형/비선형 변환방식의 문제가 아니라 특징 자체의 다양성 문제

  2. MD(Mahalanobis Distance) FPT 임계값 = 1.0 (실질적 미작동)
     - MD 정상 구간 std가 커 μ+3σ이 max를 초과
     - 정상-이상 구분이 커널 코팩스 오버랩 있음
     - 커널 공간 내 정상 군집 반경이 이동 데이터 반경과 비슷한 수준

  3. 특징 체제의 근본 문제 (특징 선별이 먼저 필요)
     - 어떤 차원 축소방식을 써도 76개 특징 중 단조성/추세성이 낮은 특징이 많으면 HI 품질이 나올 수 없음
     - 76개 → 단조성/추세성 상위 특징만 선별하면 KPCA 성능 크게 개선 기대
